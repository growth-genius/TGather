## 프로젝트 목표

- 단순 기능 구현에서 그치는 것이 아닌 실제 회사 내 운영 방식에 맞춰 구현하는 것이 목표입니다.
- TDD 방법론을 적용하여 기획 -> 테스트코드 작성 -> 개발 순으로 개발해나가고 있습니다.
- 문서화, 단위테스트는 높은 우선순위를 두어 작성했고, CI/CD를 통한 자동화를 구현하여 협업 가능한 프로젝트로 만들고 있습니다.
